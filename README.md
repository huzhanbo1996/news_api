# project #1

    Will return in json format all the negative news on baidu news. 
    
    The negative evaluation comes from [baobian](http://www.pullword.com/baobian/)

## build:

    refer to cmake

## Usage:

    api_news <port>

    curl -X GET <ip:port>

## 3rd party/Dependencies:

    rapidjson, gumbo-parser, sougou-workflow
